## Run Locally

Clone the project

```bash
  git clone https://github.com/Nierimam/Flappy-Bird-Nierimam.git
```

Go to the project directory

```bash
  cd Flappy-Bird-Nierimam
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
